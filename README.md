# Star Wars Shop

Уеб сайт за онлайн магазин със Star Wars продукти.

## Описание

Сайтът предлага книги, комикси, LEGO сетове и колекционерски фигурки от Star Wars вселената. Дизайнът е в тъмни цветове със златни акценти.

## Страници

- **index.html** - начална страница с препоръчани продукти и информация за магазина
- **products.html** - пълен каталог с филтриране по категория
- **about.html** - история на магазина и контактна информация

## Технологии

- HTML5 - семантични елементи
- CSS3 - Flexbox, Grid, media queries
- JavaScript - филтриране и модален прозорец
- Font Awesome - икони

## Функционалност

- Филтриране на продукти по категория (Книги, Комикси, LEGO, Фигурки)
- Модален прозорец с детайли при клик върху продукт
- Адаптивен дизайн за различни екрани

## Структура

```
starwars-shop/
├── index.html
├── products.html
├── about.html
├── css/
│   ├── styles.css      - импортира останалите CSS файлове
│   ├── base.css        - променливи и reset стилове
│   ├── layout.css      - header, footer, hero секция
│   ├── components.css  - карти, бутони, модал
│   └── responsive.css  - стилове за мобилни устройства
├── js/
│   └── script.js       - филтриране и модал функционалност
└── images/             - изображения на продукти
```

## Продукти

11 продукта в 4 категории:
- Книги (3) - Дарт Плегиъс, Учител и Ученик, Траун: Съюзи
- Комикси (4) - поредица Дарт Вейдър (томове 1-4)
- LEGO (2) - C-3PO, Millennium Falcon
- Фигурки (2) - Дарт Вейдър, R2-D2

## CSS архитектура

CSS е разделен на модули за по-лесна поддръжка:
- `base.css` съдържа цветовата схема като CSS променливи
- `layout.css` определя структурата на страниците
- `components.css` съдържа стилове за отделните компоненти
- `responsive.css` адаптира дизайна за мобилни устройства

## Стартиране

Отваря се index.html в браузър. Не са необходими допълнителни инструменти.
